1、查找数组元素位置
题目描述 找出元素 item 在给定数组 arr 中的位置 输出描述: 如果数组中存在 item，则返回元素在数组中的位置，否则返回 -1 示例 输入 [ 1, 2, 3, 4 ], 3 输出 2
2、数组求和
题目描述 计算给定数组 arr 中所有元素的总和 输入描述: 数组中的元素均为 Number 类型 示例 输入 [ 1, 2, 3, 4 ] 输出 10
3、计数
题目描述 统计数组 arr 中值等于 item 的元素出现的次数 示例 输入 [1, 2, 4, 4, 3, 4, 3], 4 输出 3
4、求二次方
题目描述 为数组 arr 中的每个元素求二次方。不要直接修改数组 arr，结果返回新的数组 示例 输入 [1, 2, 3, 4] 输出 [1, 4, 9, 16]
5、打印出乘法口诀表
示例
1 * 1 = 1
1 * 2 = 2    2 * 2 = 4
